Memory Game


This is a small memory game. You have 9 seconds to see the table, there are 9 three-digit numbers, and only one pair of them has the same number. Find them, and input your answer like "1,9".

You can restart the game by refreshing the page. You can also input "BIOSTAT821" to skip the game. It is better to use this page in full screen.
